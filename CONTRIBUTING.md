# 🤝 Contributing to ChatGPT Exporter

First off, thank you for considering contributing! 🎉
This project was built to solve a real problem, and contributions make it even better.

---

## 📌 Project Status

This project is **actively under development**.

* Bugs may exist 🐛
* Features are evolving 🚧
* Ideas and improvements are welcome 💡

---

## 🚀 How You Can Contribute

You can help in many ways:

* 🐛 Fix bugs
* ✨ Add new features
* 🎨 Improve UI/UX
* 📄 Improve documentation
* ⚡ Optimize performance

---

## 🛠️ Getting Started

### 1. Fork the Repository

Click the **Fork** button on GitHub.

---

### 2. Clone Your Fork

```bash
git clone https://github.com/your-username/chatgpt-exporter.git
cd chatgpt-exporter
```

---

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

Examples:

* `feature/pdf-export`
* `fix/markdown-code-block`
* `ui/improve-popup`

---

### 4. Make Changes

* Keep changes small and focused
* Follow existing code style
* Test your changes properly

---

### 5. Commit Changes

```bash
git commit -m "feat: add PDF export support"
```

Commit message examples:

* `fix: resolve markdown formatting issue`
* `feat: add TXT export option`
* `refactor: improve parsing logic`

---

### 6. Push and Create PR

```bash
git push origin feature/your-feature-name
```

Then open a **Pull Request** on GitHub.

---

## 🧪 Development Guidelines

* Write clean and readable code
* Avoid unnecessary dependencies
* Keep the extension lightweight
* Ensure compatibility with ChatGPT UI

---

## 🐛 Reporting Bugs

If you find a bug:

1. Go to **Issues** tab
2. Click **New Issue**
3. Add:

   * Description of the problem
   * Steps to reproduce
   * Expected behavior
   * Screenshots (if possible)

---

## 💡 Suggesting Features

Have an idea?

* Open an issue with label **feature request**
* Clearly explain:

  * What problem it solves
  * How it should work

---

## 🏷️ Issue Labels

We may use labels like:

* `good first issue` → beginner-friendly
* `bug` → something not working
* `enhancement` → improvements
* `feature` → new functionality

---

## 📏 Code Style

* Keep functions small and focused
* Use meaningful variable names
* Add comments where needed
* Maintain consistent formatting

---

## 🔐 Security & Privacy

* Do NOT add tracking or analytics without discussion
* Do NOT collect user data
* Respect user privacy at all times

---

## ❤️ Code of Conduct

Be respectful and constructive.

* No harassment
* No toxic behavior
* Help others learn and grow

---

## 🙌 Final Note

Even small contributions matter.

Whether it's:

* fixing a typo
* improving UI(Most welocome )
* or adding a feature

👉 It helps the project grow.

---

Thanks again for contributing! 🚀
